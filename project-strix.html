STRIX Fixed-Wing Aircraft

Scratch-Built RC Plane Designed for Manual Flight

Overview

STRIX is a manually controlled fixed-wing RC aircraft built from scratch. Inspired by agile mid-wing designs, STRIX emphasizes balance, speed, and control — all without a flight controller. The design process focused on symmetry, airfoil shaping, and CG balancing for optimal manual flight stability.

Design Features

  • Symmetrical wing layout for aerobatic control
  • Stabilized vertical and horizontal surfaces
  • Lightweight, high-thrust manual build
  • No autopilot — fully RC flown

Electronics

  • 2200KV Brushless Motor
  • 40A ESC
  • 3x 9g Servos (Elevator, Rudder, Ailerons)
  • 3S 2200mAh LiPo
  • Standard 6CH RC Receiver
