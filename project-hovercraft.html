RC Hovercraft

Dual-BLDC Powered Hovercraft Built for Surface Glide

Overview

This hovercraft was built from scratch using lightweight materials and is powered by two 1400KV brushless motors. It uses one for lift and one for forward thrust. Powered by a 1500mAh LiPo battery, the hovercraft glides smoothly on flat surfaces with full manual RC control.

Key Features

  • 2x 1400KV Brushless Motors for lift and thrust
  • Dual ESC setup with throttle mixing
  • 1500mAh 3S LiPo Battery
  • Compact and lightweight foam-based chassis
  • Stability fins for directional control
