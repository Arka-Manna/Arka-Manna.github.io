Wi-Fi Clock & Message Board

ESP8266 Smart Display with NTP Time & Web-Controlled Messaging

Overview

This simple yet smart project turns an ESP8266 and a MAX7219 LED matrix into a Wi-Fi-enabled digital clock and message board. It connects to your Wi-Fi, fetches the time using NTP, and displays it in real-time — but with a twist. A built-in web interface allows you to switch modes and show custom messages on the LED matrix.

Why I Made This

I wanted a fun, functional project that didn’t rely on apps or cloud services. With just a browser, anyone on the same Wi-Fi can control the clock or post a message. It’s the perfect DIY desk gadget.

Clock displays the text

Wiring

  • VCC → 3V
  • GND → GND
  • DIN → D7 (GPIO13)
  • CS → D8 (GPIO15)
  • CLK → D5 (GPIO14)

Web Interface

  • Shows current IP address on matrix after connecting
  • Offers buttons to switch to Clock Mode
  • Message Mode lets you enter any text to scroll on the display

    • Web interface lets you toggle between Clock and Message Mode.

Software Stack

  • ESP8266 Arduino Core
  • NTPClient for time sync
  • MD_Parola + MD_MAX72XX libraries for matrix display
  • Simple HTTP WebServer for control

Tech Stack

ESP8266 MAX7219 LED Matrix NTPClient MD_Parola WiFi WebServer Arduino IDE
