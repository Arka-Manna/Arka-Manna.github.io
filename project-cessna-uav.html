Overview

This RC plane was handcrafted from scratch without CAD, inspired by the iconic Cessna layout. It features a stable high-wing configuration, perfect for training and slow, stable flights. Built using Depron foam, the aircraft is lightweight, resilient, and efficient for manual remote-controlled flight.

Build Details

  • Airframe built from 5mm Depron foam
  • Wing span manually calculated and cut using templates
  • Control surfaces: Elevator, Rudder, Ailerons
  • Reinforced spars and lightweight bulkhead design
  • No SolidWorks — designed entirely by hand

Electronics Used

  • 920KV Brushless DC motor
  • 40A ESC for reliable throttle control
  • 3x 9g servos for flight surface control
  • Standard 6CH RC transmitter & receiver
  • 11.1V 2200mAh LiPo battery
